# 💪 Push-up – 함께 푸쉬하고 성장하는 목표 달성 앱

**Push-up**은 친구와 함께 목표를 설정하고, 푸쉬 알림을 주고받으며 매일 꾸준함을 만들어가는 챌린지 앱입니다.  
**작심삼일**을 끝내고 싶은 당신을 위해, **함께 푸쉬**하는 시스템을 제공합니다.

![push-up banner](./assets/banner.png)

---

## 🌟 핵심 기능

- 🎯 목표 설정 (제목, 기간, 주기 등)
- 👯 친구 초대 및 수락
- 🔔 나와 친구에게 푸쉬 알림 전송
- 💬 응원 메시지, 칭찬 메시지 보내기
- ✅ 목표 진행률 추적 (예: 2/4회)
